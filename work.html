My work

MyFlix Project

MyFlix is a movie application that allows users to register, login, and view a list of movies. Users can also view detailed information about each movie, including its genre and director. The application is built using React for the frontend and Node.js with Express for the backend. It also uses MongoDB for data storage and JWT for authentication.

Meet App

The Meet App is a web application that allows users to find and attend tech meetups in their area using Google Calendar's API. Users can view a list of upcoming events, and see details about each event. The application is built using React for the frontend and Node.js with Express for the backend.
